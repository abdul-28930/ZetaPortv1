Github repos that will make you an AI engineer

Curated resources to level up your AI skills—from running models locally to evaluating them and mastering prompt engineering.

    Run large language models locally and in production. Frameworks, optimizations, quantization, and deployment tools for efficient inference.

  • Awesome LLM Judges

    Evaluate and benchmark LLM outputs. LLM-as-judge setups, evaluation frameworks, and tools to measure model quality and consistency.

  • Prompt Engineering

    Learn how to write effective prompts. Techniques, patterns, and best practices to get better results from LLMs.

  • Data Science & ML Resources (veekaybee)

    A curated list of data science and machine learning resources—courses, books, and tools to build a strong foundation in AI/ML.
